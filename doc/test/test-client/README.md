# 对话式知识问答系统

一个简单的对话式知识问答系统，支持 OpenAI 兼容 API，可通过配置文件自定义 API URL 和 Key。

## 功能特点

- 🎯 **配置驱动**: 通过 YAML 配置文件自定义 API 参数
- 💬 **对话历史**: 维护多轮对话上下文
- 🧠 **记忆功能**: 自动保存对话到 `memory.md`，支持上下文记忆和自动遗忘
- 🔧 **多 API 支持**: 支持 OpenAI、阿里云百炼、LPG 代理等
- 🛡️ **PII 保护**: 可配合 LPG 代理自动脱敏敏感信息
- ⌨️ **交互友好**: 支持命令行交互，内置帮助命令

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置 API

编辑 `config.yaml` 文件，设置您的 API 信息：

```yaml
api:
  base_url: "http://127.0.0.1:8080"  # LPG 代理地址
  api_key: "your-virtual-key"         # LPG 虚拟 Key
  model: "qwen3.6-plus"               # 模型名称
```

### 3. 运行程序

```bash
python chatbot.py
```

## 配置说明

### API 配置

| 参数 | 说明 | 示例 |
|------|------|------|
| `base_url` | API 基础 URL | `http://127.0.0.1:8080` |
| `api_key` | API Key | `sk-virtual-...` |
| `model` | 模型名称 | `qwen3.6-plus` |
| `timeout` | 请求超时（秒） | `30` |

### 支持的 API 提供商

1. **LPG 代理**
   ```yaml
   base_url: "http://127.0.0.1:8080"
   api_key: "sk-virtual-..."
   model: "qwen3.6-plus"
   ```

2. **阿里云百炼**
   ```yaml
   base_url: "https://dashscope.aliyuncs.com/compatible-mode/v1"
   api_key: "sk-..."
   model: "qwen3.6-plus"
   ```

3. **OpenAI**
   ```yaml
   base_url: "https://api.openai.com/v1"
   api_key: "sk-..."
   model: "gpt-3.5-turbo"
   ```

## 使用说明

### 基本对话

直接输入问题，系统会返回答案：

```
👤 你: 什么是人工智能？

🤖 助手: 人工智能（AI）是计算机科学的一个分支，致力于创建能够执行通常需要人类智能的任务的系统...
```

### 内置命令

| 命令 | 说明 |
|------|------|
| `/clear` | 清空对话历史 |
| `/memory` | 显示记忆统计 |
| `/forget` | 清空所有记忆 |
| `/config` | 显示当前配置 |
| `quit` | 退出程序 |

### 对话历史管理

系统会自动维护对话上下文，支持多轮问答。当历史消息超过配置的最大数量时，会自动清理旧消息。

### 记忆功能

系统会自动将对话保存到 `memory.md` 文件：

- **自动保存**: 每次对话后自动追加到记忆文件
- **上下文记忆**: 启动时自动加载最近的对话作为上下文
- **自动遗忘**: 当记忆文件超过 1000 行时，自动清理较早的对话
- **持久化**: 记忆文件保存在本地，程序重启后依然保留

记忆文件格式示例：

```markdown
# 对话记忆

## 2026-04-17 21:35:00 | 模型: qwen3.6-plus

**用户**: 你好

**助手**: 你好！有什么我可以帮你的吗？

---

## 2026-04-17 21:36:00 | 模型: qwen3.6-plus

**用户**: 什么是人工智能？

**助手**: 人工智能（AI）是计算机科学的一个分支...

---
```

## 与 LPG 集成

当配合 LLM Privacy Gateway 使用时，系统会自动脱敏敏感信息：

```
👤 你: 我的邮箱是 test@example.com

🤖 助手: 我已收到您的信息（邮箱已脱敏为 <EMAIL_ADDRESS>）...
```
