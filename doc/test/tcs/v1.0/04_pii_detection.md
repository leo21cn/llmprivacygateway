# LLM Privacy Gateway v1.0 PII检测脱敏黑盒测试用例

**文档版本：** 1.0  
**创建日期：** 2026-04-04  
**测试类型：** 黑盒功能测试  
**覆盖范围：** Presidio集成模块 - PII检测与脱敏功能  

---

## 1. 测试概述

本文档覆盖 LLM Privacy Gateway v1.0 版本 Presidio 集成模块的 PII（个人身份信息）检测与脱敏功能黑盒测试用例，包括检测准确性、脱敏策略、多语言支持、边界情况等测试。

### 1.1 测试环境

| 项目 | 说明 |
|------|------|
| 操作系统 | macOS / Linux |
| Python 版本 | 3.9+ |
| Presidio 版本 | 2.2+ |
| 测试工具 | 手动执行 / 自动化脚本 |
| 前置依赖 | Presidio Analyzer & Anonymizer 服务运行中 |

### 1.2 测试数据

| 文件路径 | 说明 |
|----------|------|
| `test_data/pii_samples/email_samples.txt` | 邮箱地址样本 |
| `test_data/pii_samples/phone_samples.txt` | 手机号样本 |
| `test_data/pii_samples/idcard_samples.txt` | 身份证号样本 |
| `test_data/pii_samples/creditcard_samples.txt` | 信用卡号样本 |
| `test_data/pii_samples/name_samples.txt` | 人名样本（中英文） |
| `test_data/pii_samples/address_samples.txt` | 地址信息样本 |
| `test_data/pii_samples/mixed_pii_samples.txt` | 混合PII内容样本 |
| `test_data/pii_samples/multilingual_samples.txt` | 多语言文本样本 |
| `test_data/pii_samples/edge_cases.txt` | 边界情况样本 |
