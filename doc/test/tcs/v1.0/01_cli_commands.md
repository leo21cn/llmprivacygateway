# LLM Privacy Gateway v1.0 CLI 命令黑盒测试用例

**文档版本：** 1.0  
**创建日期：** 2026-04-04  
**测试类型：** 黑盒功能测试

---

## 1. 测试概述

本文档覆盖 LLM Privacy Gateway v1.0 版本所有 CLI 命令的黑盒测试用例，包括正常流程和异常流程测试。

### 1.1 测试环境

| 项目 | 说明 |
|------|------|
| 操作系统 | macOS / Linux |
| Python 版本 | 3.9+ |
| 测试工具 | 手动执行 / 自动化脚本 |
| 前置依赖 | Presidio 服务（可选） |

### 1.2 测试数据

| 文件路径 | 说明 |
|----------|------|
| `test_data/config_sample.yaml` | 标准配置文件样本 |
| `test_data/config_invalid.yaml` | 格式错误的配置文件 |
| `test_data/providers.yaml` | 提供商配置样本 |
| `test_data/rules_sample.yaml` | 规则文件样本 |
| `test_data/test_text.txt` | 测试用文本文件 |
