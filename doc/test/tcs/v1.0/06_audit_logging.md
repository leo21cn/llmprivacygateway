# LLM Privacy Gateway v1.0 审计日志黑盒测试用例

## 概述

本文档包含LLM Privacy Gateway v1.0审计日志模块的黑盒测试用例，覆盖日志记录、查询、统计、导出、清理、格式和性能等功能。

## 1. 日志记录

### TC-AUDIT-001: 记录正常请求日志

| 项目 | 内容 |
|------|------|
| **用例ID** | TC-AUDIT-001 |
| **用例名称** | 记录正常请求日志 |
| **前置条件** | 1. LLM Privacy Gateway已启动<br>2. 审计日志功能已启用<br>3. 日志级别设置为info |
| **测试步骤** | 1. 发送一个正常的LLM请求到代理服务器<br>2. 等待请求完成<br>3. 查询审计日志 |
| **测试数据** | 有效的LLM请求数据（参考02_proxy_service_testdata.md） |
| **预期结果** | 1. 日志中包含完整的请求信息<br>2. 日志级别为info<br>3. 记录请求时间、客户端IP、请求路径、状态码、耗时<br>4. 日志格式符合JSON规范 |
| **优先级** | P0 |
| **自动化** | 是 |
