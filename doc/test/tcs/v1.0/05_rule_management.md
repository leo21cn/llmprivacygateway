# LLM Privacy Gateway v1.0 规则管理黑盒测试用例

**文档版本：** 1.0  
**创建日期：** 2026-04-04  
**测试类型：** 黑盒功能测试  
**覆盖范围：** 规则管理模块  

---

## 1. 测试概述

本文档覆盖 LLM Privacy Gateway v1.0 版本规则管理模块的黑盒测试用例，包括规则加载、列表、启用/禁用、导入、移除、测试、配置、优先级和持久化等功能。

### 1.1 测试环境

| 项目 | 说明 |
|------|------|
| 操作系统 | macOS / Linux |
| Python 版本 | 3.9+ |
| 测试工具 | 手动执行 / 自动化脚本 |
| 前置依赖 | LPG 核心模块 |

### 1.2 测试数据

| 文件路径 | 说明 |
|----------|------|
| `test_data/rules_builtin/` | 内置规则目录 |
| `test_data/rules_custom/` | 自定义规则目录 |
| `test_data/rules_sample.yaml` | 规则文件样本（YAML格式） |
| `test_data/rules_sample.json` | 规则文件样本（JSON格式） |
| `test_data/rules_invalid.yaml` | 格式错误的规则文件 |
| `test_data/rules_empty.yaml` | 空规则文件 |
| `test_data/rules_duplicate.yaml` | 重复规则文件 |
| `test_data/config_with_rules.yaml` | 包含规则配置的配置文件 |
| `test_data/test_text.txt` | 测试用文本文件 |

## 3. 附录

### 3.1 测试命令参考

| 命令 | 说明 |
|------|------|
| `lpg rule list` | 列出所有规则 |
| `lpg rule list --category <category>` | 按分类列出规则 |
| `lpg rule list --enabled` | 列出启用的规则 |
| `lpg rule list --disabled` | 列出禁用的规则 |
| `lpg rule enable <rule_name>` | 启用规则 |
| `lpg rule disable <rule_name>` | 禁用规则 |
| `lpg rule import <file>` | 导入规则文件 |
| `lpg rule remove <rule_name>` | 移除规则 |
| `lpg rule test <rule_name> --text <text>` | 测试规则 |
| `lpg rule config` | 显示规则配置 |

### 3.2 规则分类

| 分类 | 说明 | 示例 |
|------|------|------|
| pii | 个人身份信息 | 姓名、邮箱、电话、地址 |
| credentials | 凭证信息 | 密码、API Key、Token |
| finance | 金融信息 | 信用卡号、银行账号、交易记录 |

### 3.3 规则状态

| 状态 | 说明 |
|------|------|
| enabled | 规则启用，参与检测 |
| disabled | 规则禁用，不参与检测 |

### 3.4 规则优先级

| 优先级 | 说明 | 值 |
|--------|------|-----|
| 高 | 优先应用 | 1-100 |
| 中 | 普通应用 | 101-200 |
| 低 | 最后应用 | 201-300 |
